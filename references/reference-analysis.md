# Reference Analysis

Use this file when the user provides reference images, links, screenshots, or attached visuals.

## What to extract

For each reference, note only the traits that matter to prompt quality:

- composition and framing
- camera angle or perspective
- detail density
- shape language
- palette and contrast
- lighting direction and intensity
- material feel or texture
- mood
- background treatment
- what should be avoided

## Questions to ask

If the user provides references but not enough guidance, ask:

- What exactly do you want to borrow from each ref?
- Which ref is the primary anchor?
- What should stay out of the final image even if it appears in the refs?
- Do you want the final image to feel closer to ref A, ref B, or a blend?

## Synthesis pattern

Before drafting the prompt, summarize:

1. shared traits across the refs
2. meaningful differences
3. the chosen primary anchor
4. the final art direction you will carry into the prompt

Do not claim the refs are coherent if they are not.
